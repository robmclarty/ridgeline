Usage: ridgeline dry-run [options] [build-name]

Display the plan without executing

Options:
  --model <name>        Model for planner (defaults to settings.json model, or
                        'opus')
  --timeout <minutes>   Max duration for planning (default: "120")
  --constraints <path>  Path to constraints.md
  --taste <path>        Path to taste.md
  -h, --help            display help for command
