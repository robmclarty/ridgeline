Usage: ridgeline retro-refine [options] [build-name]

Produce a refined version of the original input spec from learnings.md and the
just-completed build's artifacts. Writes refined-input.md to the build
directory.

Options:
  --model <name>       Model for retro-refiner agent (defaults to settings.json
                       model, or 'opus')
  --timeout <minutes>  Max duration in minutes (default: "10")
  --specialists <n>    Number of ensemble specialists (1, 2, or 3). Default: 3
  --thorough           Alias for --specialists 3 (the default)
  -y, --yes            Skip the preflight confirmation prompt
  --auto               Non-interactive mode: skip Q&A; use Auto variants where
                       applicable
  -h, --help           display help for command
