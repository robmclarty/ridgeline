# Helloworld Example

A minimal ridgeline example that builds a single `hello.js` script.

## Prerequisites

Ridgeline must be built first. From the repository root:

```bash
npm install
npm run build
```

## What's Here

- `hello.js` — the generated implementation (a `greet(name)` function)
- `package.json` — minimal project metadata
- `.ridgeline/builds/helloworld/` — the build artifacts from a completed run:
  - `shape.md` — project context gathered by the shaper
  - `spec.md` — what to build (features, acceptance criteria)
  - `constraints.md` — technical guardrails (language, check command)
  - `snapshot.md` — directory tree and config files at time of shaping
  - `phases/` — phase specs generated by the planner
  - `state.json` — pipeline and phase statuses, retries, timestamps
  - `budget.json` — per-invocation cost tracking
  - `trajectory.jsonl` — append-only event log
  - `handoff.md` — accumulated context from prior phases

## Running the Example

Run the generated script:

```bash
node hello.js
# Hello, World!
```

## Re-running from Scratch

To re-run the full ridgeline pipeline on this example, first rewind the state
and then run the build. All commands should be run from this directory.

```bash
# Rewind to the beginning (clears spec, plan, and build artifacts)
node ../../dist/cli.js rewind helloworld --to shape

# Remove the generated output
rm -f hello.js

# Re-run the full pipeline (shape → spec → plan → build)
node ../../dist/cli.js helloworld
```

You can also run individual stages:

```bash
node ../../dist/cli.js shape helloworld
node ../../dist/cli.js spec helloworld
node ../../dist/cli.js plan helloworld
node ../../dist/cli.js build helloworld
```
