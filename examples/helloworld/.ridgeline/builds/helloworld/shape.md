# Shape: helloworld

## What the user wants

A simple "hello world" Node.js script that defines a reusable greeting function and prints a greeting when run directly.

## Project context

- Bare Node.js project with only a `package.json` in place
- No existing source files, no dependencies, no build tooling
- Language: JavaScript (CommonJS)

## Key decisions

- The script should work both as a standalone executable and as an importable module
- No external dependencies required
- No TypeScript or build step
